About the game

Froget is a single-player top-down game created for the Global Game Jam 21. It was created by 2 people within 48 hours using the unity game engines universal render pipeline, allowing for a 2D game to be made in a 3D space meaning unity features such as particle effects and lighting could be used. In the game, you play as a forgetful frog who need to adventure around the tile-based map killing ghosts and find his keycaps which he keeps miss placing. The Jams topic was lost and found the original goal was to make the keycaps affect the amount of functionality the player would have. Unfortunately this could not be made before the end of the Jam.
